---
title: "Connectivity Displays Incorrectly"
description: "To troubleshoot connectivity tier issues, users can test network speeds via a sample Docker container. Here's how:"
---

1. Pull the Python 3.9 Slim container:

   ```
   docker pull python:3.9-slim
   ```
2. Run the container:

   ```
   docker run -it --name speedtest-container python:3.9-slim /bin/bash
   ```
3. Install the speedtest tool:

   ```
   pip install speedtest-cli
   ```
4. Test the network speed:

   ```
   speedtest-cli
   ```
